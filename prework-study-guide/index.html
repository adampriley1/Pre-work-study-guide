HTML

  • The head element contains infomration about the webpage
  • The body element represents the visible content shown to the user
  • Header elements help give structure and order to the user
  • Headers must be used in sequence H1 > H6
  • Paragraph elements are for the main content
  • Images can be added, but need a source and alt text

CSS

  • CSS gives style to the HTML text to improve appearance
  • Css consists of a selector, propery and value
  • CSS can be an Inline, Internal or External CSS Style sheet
  • Items can be targetted by selectors. These can be ID, class or element
  • * is a selector that applies to all elements on a page
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • Git is a version control system
  • It allows mulitple users to work on the same project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git status shows modified files in the working directory
  • git add - adds a file to next commit

JavaScript

  • JavasScript makes a webpage interactive for the user
